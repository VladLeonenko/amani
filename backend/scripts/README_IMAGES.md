# Инструкция по загрузке изображений товаров

## Проблема
Изображения товаров ссылаются на `http://localhost:3845`, который недоступен. Нужно загрузить изображения в БД и привязать их к товарам.

## Решение

### Вариант 1: Автоматическая обработка из папки (рекомендуется)

1. **Создайте папку для изображений:**
   ```bash
   mkdir -p frontend/public/products
   ```

2. **Поместите изображения товаров в эту папку:**
   - Назовите файлы по slug товара (можно посмотреть в админ панели или в БД)
   - Поддерживаемые форматы: `.jpg`, `.jpeg`, `.png`, `.webp`

3. **Примеры имен файлов:**
   - `африканская-маска-дух-предков.jpg`
   - `картина-саванна-на-закате.jpg`
   - `скульптура-воин-масаи.jpg`
   - `постер-африканские-ритмы.jpg`
   - `фотография-жирафы-в-саванне.jpg`
   - `картина-африканская-деревня.jpg`
   - `скульптура-мать-и-дитя.jpg`
   - `картина-золотой-закат.jpg`
   - `фотография-слоны-в-национальном-парке.jpg`

4. **Запустите скрипт обработки:**
   ```bash
   cd backend
   node scripts/process-product-images-from-folder.js
   ```

5. **Скрипт автоматически:**
   - Найдет все изображения в папке `frontend/public/products/`
   - Оптимизирует их (ресайз до 1200px, сжатие)
   - Сохранит в `backend/uploads/products/`
   - Обновит ссылки в БД

### Вариант 2: Загрузка через админ панель

1. Откройте админ панель
2. Перейдите в раздел "Продукты и стоимость"
3. Откройте каждый товар для редактирования
4. Загрузите изображение через интерфейс загрузки
5. Сохраните товар

### Вариант 3: Прямая загрузка через API (для разработчиков)

Если у вас есть изображения в другом месте, можно использовать API:

```bash
curl -X POST http://localhost:3000/api/images \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "img=@/path/to/image.jpg"
```

## Проверка результата

После обработки проверьте:
1. Изображения должны быть в `backend/uploads/products/`
2. В БД товаров поле `image_url` должно содержать путь вида `/uploads/products/...`
3. На сайте изображения должны отображаться корректно

## Структура папок

```
amani/
├── frontend/
│   └── public/
│       └── products/          # ← Поместите исходные изображения сюда
│           ├── африканская-маска-дух-предков.jpg
│           ├── картина-саванна-на-закате.jpg
│           └── ...
└── backend/
    └── uploads/
        └── products/          # ← Сюда скрипт сохранит обработанные изображения
            ├── африканская-маска-дух-предков-1234567890-abc123.jpg
            └── ...
```

## Примечания

- Скрипт автоматически оптимизирует изображения (ресайз до 1200px, сжатие)
- Исходные файлы остаются в `frontend/public/products/`
- Обработанные файлы сохраняются в `backend/uploads/products/`
- Если изображение не найдено, скрипт покажет ожидаемое имя файла
